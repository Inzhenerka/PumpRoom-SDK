PumpRoom SDK

Лёгкая библиотека для интеграции LMS с PumpRoom.

Установка

npm install pumproom-sdk

Или подключите UMD-бандл:

<script src="bundles/pumproom-sdk-latest.umd.js"></script>

Пример использования

import { init, authenticate } from 'pumproom-sdk';

init({ apiKey: 'API_KEY', realm: 'inzh', cacheUser: true });

authenticate(getLmsProfile());

Исходный код и примеры смотрите в репозитории.
