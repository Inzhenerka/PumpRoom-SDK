PumpRoom SDK

Простой способ подключить PumpRoom к LMS

Установка

Скачайте пакет из npm или подключите UMD-бандл:

npm install pumproom-sdk
<script src="bundles/pumproom-sdk-latest.umd.js"></script>

Использование ES-модуля

import { init, authenticate } from 'pumproom-sdk';

init({ apiKey: 'API_KEY', realm: 'inzh', cacheUser: true });
authenticate(profileObj);

Использование через <script>

<script src="bundles/pumproom-sdk-latest.umd.js"></script>
<script>
PumpRoomSdk.init({ apiKey: 'API_KEY', realm: 'inzh', cacheUser: true });
PumpRoomSdk.authenticate(profileObj);
</script>

Подробные примеры находятся в каталоге example.
