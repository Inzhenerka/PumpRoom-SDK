PumpRoom SDK

Интеграция LMS с PumpRoom

Библиотека упрощает авторизацию и обмен сообщениями между вашим сайтом и платформой PumpRoom.

Быстрый старт

Установите пакет и подключите его в проекте.

npm install pumproom-sdk
import { init, authenticate } from 'pumproom-sdk';

init({ apiKey: 'API_KEY', realm: 'inzh', cacheUser: true });
authenticate(profileObj);

Добавьте UMD-бандл в вашу страницу.

<script src="https://pumproom-sdk.inzhenerka-cloud.com/bundles/pumproom-sdk-v__VERSION__.umd.js"></script>
<script>
  PumpRoomSdk.init({ apiKey: 'API_KEY', realm: 'inzh', cacheUser: true });
  PumpRoomSdk.authenticate(profileObj);
</script>

Методы SDK

Метод Аргументы Описание
init(config) apiKey, realm, cacheUser?, minHeight? Инициализация SDK и установка конфигурации.
authenticate(profile) profile – объект пользователя Авторизация пользователя и кэширование токена.
getCurrentUser() нет Возвращает текущего авторизованного пользователя.
handleFullscreenToggle() нет Восстанавливает прокрутку после выхода из полноэкранного режима.
enforceIframeHeight(minHeight) minHeight – минимальная высота Устанавливает минимальную высоту iframe PumpRoom.
getVersion() нет Возвращает текущую версию библиотеки.

Примеры

ES-модуль

import { init, authenticate } from 'pumproom-sdk';

init({ apiKey: 'KEY', realm: 'inzh' });
authenticate(profileObj);

UMD-бандл

<script src="https://pumproom-sdk.inzhenerka-cloud.com/bundles/pumproom-sdk-v__VERSION__.umd.js"></script>
<script>
  PumpRoomSdk.init({ apiKey: 'KEY', realm: 'inzh' });
</script>

Настройка высоты iframe

import { enforceIframeHeight } from 'pumproom-sdk';

enforceIframeHeight(600);
